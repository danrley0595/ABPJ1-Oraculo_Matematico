# ABPJ1-Planejamento_Financeiro

## Objetivo Projeto

Esse projeto tem como por objetivo ajuar no controle financeiro, permitindo:

- Informar o sálario como receita
- informar as despesas:
    - Aluguel
    - Alimentação
    - Cartão
    - lazer
  - Somar todas as despesas
  - Calcular o saldo restante(Com ou sem meta)
  - Informar a situação financeira(Saldo negativo, zero, positivo)
  - Informar Valor da meta e valor mensal que deseja par aatingir a meta
  - Calcular

## Como usar
1. Informar o valor do sálario
2. Informar o valor de cada despesa
3. O programa mostrará a situação financeira e o saldo restante sem o valor da meta mensal
4. Informar o valor da meta desejada
5. Informar o valor da meta mensal
6. O programa mostrará a quantidade de meses necessarios para atingimento da meta
7. O programa mostrará a situação financeira e o saldo restante com o valor da meta mensal
